## ⚙️ Setup

### Run Locally
No build step needed — just open in browser:
```bash
git clone https://github.com/yourname/portfolio.git
cd portfolio
# Open index.html in your browser
```

### Add Your Photo
Place your photo in the `assets/` folder and name it `profile.jpg`.

### Update Links
Before going live, replace these placeholders in `index.html`:

| Placeholder | Replace with |
|---|---|
| `github.com/yourname` | Your GitHub username |
| `https://drive.google.com/your-resume-link` | Your Google Drive resume link |
| `https://drive.google.com/your-resume-preview-link` | Your Google Drive preview embed link |

---

## 🌐 Deploy to GitHub Pages

```bash
git init
git add .
git commit -m "initial commit"
git remote add origin https://github.com/yourname/portfolio.git
git push -u origin main
```

Then go to:  
**GitHub Repo → Settings → Pages → Source: main branch / root**

Your site will be live at `https://yourname.github.io/portfolio` within a few minutes.
